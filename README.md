# Cadastro de Alunos – Estrutura de Dados

Este repositório contém um programa desenvolvido em **C++** para a disciplina de **Estrutura de Dados**.  
O sistema realiza o cadastro de alunos utilizando `struct` e `std::vector`, permitindo o armazenamento de informações como nome, sexo, idade e notas.

A aplicação percorre os dados cadastrados, calcula a média final de cada aluno e determina automaticamente sua situação acadêmica, classificando-o como **aprovado** ou **reprovado** de acordo com o critério de média adotado.

O projeto tem como objetivo reforçar conceitos fundamentais da linguagem C++ e o uso de estruturas de dados básicas no contexto acadêmico.
